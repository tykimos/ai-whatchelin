## 주요 변경사항

- **Google Gemini 3.1 Flash Live** (3월 26일): 음성 우선 멀티모달 모델 발표, Search Live를 200개국 이상으로 확장. 향상된 소음 처리, 도구 사용, 대화 품질.
- **Claude Code 모멘텀**: Anthropic은 52일(2월 3일~3월 24일)간 74회 릴리스를 출시, Claude Code의 ARR이 25억 달러를 돌파함.
- **DeepSeek V4 지연**: 원래 3월 초 예정이었으나 2026년 4월로 연기.
- **Windsurf 가격 정책 개편**: 3월 19일에 크레딧에서 일간/주간 할당량으로 전환, 커뮤니티 반발 발생.
