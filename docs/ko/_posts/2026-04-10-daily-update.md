## 주요 변경사항

- **Claude Mythos 미리보기** (4월 7일): Anthropic이 Project Glasswing을 통해 제한적인 Mythos 미리보기를 공식 출시 — Amazon, Apple, Microsoft, CrowdStrike 등 12개 파트너가 방어적 보안에 배포 중. 약 10조 파라미터 모델.
- **Cursor 3** (4월 2일): AI 에이전트 중심으로 재건된 주요 릴리스 — 로컬/SSH/클라우드 전반의 병렬 에이전트를 위한 에이전트 창, 자연어 UI 편집을 위한 디자인 모드, 멀티 LLM 비교.
- **Cursor Bugbot** (4월 8일): MCP 지원과 함께 실시간 자기 개선, 해결률 80% 근접.
- **Copilot VS Code Autopilot** (4월 8일): 공개 미리보기에서 완전 자율 에이전트 세션, 채팅에서 이미지/비디오 지원, 구성 가능한 사고 노력.
- **Copilot CLI v1.0.21** (4월 7일): MCP 서버 관리를 위한 새로운 `copilot mcp` 명령어.
- **GitHub Mobile** (4월 8일): 서명된 커밋과 함께 모바일에서 Copilot 클라우드 에이전트 이용 가능.
- **Gemini CLI v0.37.0** (4월 8일): 동적 샌드박스 확장, 챕터(도구 기반 주제 그룹화), 영구 세션을 갖춘 향상된 브라우저 에이전트.
- **ChatGPT Apple CarPlay**: iOS 26.4+ 지원 차량에서 핸즈프리 대화 제공.
- **Cursor 3 출시 (4월 2일):** 저장소 전반의 병렬 에이전트를 위한 에이전트 창, 브라우저에서 UI 주석을 위한 디자인 모드, /worktree, /best-of-n, Bugbot 78% 해결률
- **Gemini CLI v0.37.0 (4월 8일):** 동적 샌드박스 확장, "챕터" 도구 기반 주제 그룹화, 영구 세션을 갖춘 향상된 브라우저 에이전트
- **GitHub Copilot Autopilot 미리보기 (4월 9일):** 수동 승인 없는 완전 자율 에이전트 세션
- **Copilot SDK 공개 미리보기 (4월 2일):** 맞춤형 앱에 Copilot 에이전트 기능 내장
- **Copilot CLI v1.0.22 (4월 9일):** MCP 스키마 정제, 대용량 이미지 처리 개선, 새로운 인라인 렌더러
- **ChatGPT File Library + CarPlay:** 재사용을 위한 파일 자동 저장, iOS 26.4+에서 핸즈프리 접근
- **GPT-5.3-Codex 업그레이드:** 25% 빠른 속도, 강화된 추론, 실시간 진행 상황 업데이트
- **Gemma 4 출시 (4월 2일):** 31B Dense로 Arena AI 3위, Apache 2.0
