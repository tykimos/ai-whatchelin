## 주요 변경사항

- **ChatGPT 5.5 + SuperApp 출시 (4월 6일)**: OpenAI가 ChatGPT, Codex, Atlas 브라우저를 하나의 데스크톱 앱으로 통합. 두 README 모두의 타임라인, 릴리스 로그, 킬러 기능, 랭킹 설명 업데이트.
- **Gemini 인터랙티브 3D 시각화 (4월 10일)**: Google Gemini가 이제 채팅에서 직접 인터랙티브 3D 모델, 물리 시뮬레이션, 조정 가능한 차트를 생성할 수 있음 (Pro 모델). 타임라인, 릴리스 로그, 사실 확인 로그에 추가됨.
- **Sora 종료 일정 명확화**: 앱은 4월 26일 종료, API는 9월 24일 종료. 종료 설명 및 릴리스 로그 항목이 정확한 날짜로 업데이트됨.
- **Codex CLI v1.0.23** (4월 10일): 직접 에이전트 모드 시작을 위한 `--mode`, `--autopilot`, `--plan` 플래그 추가
- **Copilot Pro 체험 일시 중지** (4월 10일): GitHub가 남용으로 인해 신규 체험을 중지; 기존 체험/구독은 영향 없음
- **Anthropic, OpenClaw 차단** (4월 10일): Claude 구독 접근이 OpenClaw 같은 서드파티 도구에 차단됨; 사용자는 최대 50배 비용 증가; OpenClaw 제작자 일시 차단
- **Claude Code v2.1.98** (4월 9일): /team-onboarding, 원격 세션 자동 설정, MCP 500K 문자 제한, CLI에서 computer use
- **Gemini CLI v0.37.1** (4월 9일): 동적 샌드박스 및 워크트리 지원 버그 수정
