## 주요 변경사항

- **Cursor Composer 2 / Kimi K2.5 스캔들**: Cursor의 Composer 2 모델이 Moonshot AI의 오픈소스 Kimi K2.5를 출처 표기 없이 기반으로 삼은 것이 밝혀졌습니다. 기업가치 500억 달러의 Cursor는 개발자가 API 응답에서 `kimi-k2p5` 모델 ID를 발견한 후 베이스 모델을 인정했습니다. 두 README 모두의 IDE 커뮤니티 반응 섹션에 추가됨.
- **Sora 서비스 종료 (2026.03.24)**: OpenAI가 Sora 앱과 API를 종료했습니다. 두 README 전체의 크리에이터 섹션(영상 테이블, 플로우차트, 가격 레이더, 순위)에 종료 표시.
- **Cursor 기업가치**: 290억 달러에서 500억 달러 규모의 협상으로 업데이트 (Bloomberg, 3월 12일). ARR 20억 달러 이상 확인.
- **Antigravity**: 지속적인 할당량 논란 속에 인기도 계속 하락 (61, 62에서 하락).
- **Gemini CLI 무료 티어, Flash 모델만 이용 가능** (2026년 3월 25일부터 시행). 무료 사용자는 CLI에서 Gemini Pro 모델에 더 이상 접근 불가 — Google AI Pro/Ultra 구독 필요.
- **Windsurf 가격 정책 개편**: 크레딧이 일간/주간 할당량으로 교체. Pro 가격이 월 15달러에서 20달러로 인상. 기존 15달러 구독자는 기존 가격 유지.
- **Gemini 3.1 Pro, GitHub Copilot에서 이용 가능** (공개 미리보기, 3월 23일) — VS Code, JetBrains, Xcode, Eclipse 지원.
