## 주요 변경사항

- **Cursor 3.0 출시 (4월 2일)**: 회사 역사상 최대 업데이트 — 병렬 에이전트 실행을 위한 에이전트 창, 브라우저 UI 주석을 위한 디자인 모드, 에이전트 탭, Git 격리를 위한 /worktree, 멀티모델 비교를 위한 /best-of-n
- **Copilot SDK 공개 미리보기 (4월 2일)**: GitHub가 Copilot 클라우드/CLI 뒤의 동일한 에이전트 런타임을 노출하는 Copilot SDK 출시. Node.js, Python, Go, .NET, Java 지원
- **Gemini CLI의 Gemini 3 Flash**: 2.5 Pro보다 3배 빠르고 비용 절감. Gemini 3.1 모델도 지원
- **Claude Code /powerup 학습**: 애니메이션 데모로 Claude Code 기능을 가르치는 새로운 인터랙티브 학습 기능
- **Codex CLI 개선**: Windows 샌드박스 프록시 전용 네트워킹, ChatGPT 디바이스 코드 로그인, MCP 시작 안정성 향상
